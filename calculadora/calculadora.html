Configuracion Odoo 19

Conecta tu Odoo para consultar stock y ventas. Necesitas generar un API Key en tu perfil de Odoo.

Como obtener el API Key

1. Entra en tu Odoo (paracarpinteros.odoo.com)
2. Arriba derecha → tu nombre → Mi Perfil
3. Pestaña "Seguridad de la Cuenta"
4. "API Keys" → "New API Key"
5. Ponle nombre "Calculadora" y copia la clave
6. Pegala aqui arriba
